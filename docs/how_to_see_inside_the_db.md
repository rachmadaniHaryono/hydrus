# How To See Inside The DB

The databases are sqlite. You can look inside them with many programs. I like to use:

- [SQLiteStudio](http://sqlitestudio.pl/)
- [The SQLite command-line shell program](https://www.sqlite.org/download.html)

I include the command-line shell with the normal frozen Windows releases.

Feel free to explore how hydrus works, and if you have any questions or comments, don't hesitate to [contact me](contact.md)!
